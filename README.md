# nijisanji-stream-playlist-creator

- にじさんじの配信情報から、特に自分好みの配信だけを抜き出してプレイリスト化するスクリプトです。
- 特に雑談などのラジオ系配信、かつメン限などを除いたものを中心に、登録済みチャンネル、配信中のものを先頭でピックアップします。
